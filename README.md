# Appnavn og forfatter

- **Navn:** Snack Bar
- **Forfatter:** Ola Haug Letrud

## Om Snack Bar

En enkel React-applikasjon laget som arbeidskrav i emnet Javascript Rammeverk ved Gokstad Akademiet. Applikasjonen viser en liste med produkter og lar brukeren fjerne varer fra listen – en liten demo på state-håndtering i React.

## Installering og oppstart 

```bash
npm install
npm run dev
```

## Funksjonalitet

-**Viser en liste med produkter**

-**Mulighet til å fjerne produkter fra listen**

-**Enkel styling og layout**
